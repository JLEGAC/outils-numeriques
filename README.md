# Compteur de lignes (GitHub Pages)

Petit site 100% statique (HTML/CSS/JS) pour compter les lignes d'un texte.

## Mise en ligne sur GitHub Pages

1. Crée un repo GitHub (ex: `compteur-lignes`).
2. Ajoute les fichiers : `index.html`, `styles.css`, `script.js`.
3. Va dans **Settings → Pages**.
4. **Source** : `Deploy from a branch`.
5. **Branch** : `main` (ou `master`) / **Folder** : `/ (root)`.
6. Sauvegarde : tu obtiens une URL du type `https://<ton-user>.github.io/compteur-lignes/`.

## Usage local

Ouvre `index.html` dans ton navigateur — ça marche hors-ligne.
